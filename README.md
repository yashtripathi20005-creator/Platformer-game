# Phaser Platformer Game

A complete platformer game built with Phaser.js 3 and served with a Python Flask backend.

## Features

- 🎮 Smooth player controls (arrow keys)
- 🪙 Collectible coins with score tracking
- 👾 Enemy AI with patrol behavior
- ❤️ Lives system (3 lives)
- 🏆 Victory condition (collect all coins)
- 🎨 Beautiful visual design with animations
- 📱 Responsive UI overlay

## Controls

- **Arrow Left**: Move left
- **Arrow Right**: Move right
- **Arrow Up**: Jump (double jump supported!)
